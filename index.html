Option in Licence at University Paris 8.

Semester 1:

Creation and management of online resources

The objective of this course is to get started with software for creating and editing graphic resources (for example with Gimp, Inkscape, Krita, Synfig, Blender, or even PikoPixel) and sound (for example with Audacity, Ardor, or even LMMS), as well as the management of these resources as part of the design of a software project (typically, a Godot project).

Pratice with the machines

The objective of this course is to become familiar with the GNU / Linux environment, the command line and to get to know a little more closely the machines that will be used throughout the training.

Introduction the game engine

The objective of this course is to introduce the functioning of game engines (their role, their functionalities), as well as the practical handling of a game engine in particular (Godot).

Basic Computing

The objective of this course is to introduce through practical programming exercises, some essential theoretical notions of computer science: Boolean logic, sets and functions, enumeration, automata and Turing machine, cellular automata, computability, complexity, as well as basics of algebra.

Functional Programming

The objective of this course is to learn programming. The objective of this course is to learn the best practices to adopt to pay attention to your online image, as well as to create and maintain a simple personal web page that will be fed by the various academic and personal projects of each one. .

Programming Methodology

The objective of this course is to learn how to learn to program. It is a question of acquiring the bases essential to the learning of the programming and thus to the pursuit of the license. This course is an introduction to the use of several programming languages(Python, C, Bash, LaTeX), as well as some essential tools (interpreters, compilers, version control manager, compilation manager). Each point will be reviewed and deepened later in other courses of the license.
